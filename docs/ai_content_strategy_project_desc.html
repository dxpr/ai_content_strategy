This module is part of the AI module ecosystem and included in DXPR CMS.

Transform your content gaps into growth opportunities. AI Content Strategy analyzes your existing content and provides strategic content recommendations tailored to your site to strengthen your market position.

You need AI Content Strategy if

  • Your content marketing feels scattered or lacks strategic direction
  • You struggle to identify what content to create next
  • You want to establish thought leadership in your industry
  • You need data-driven content recommendations based on your existing assets

Features

  • Configurable recommendation categories:
    • Customizable category types with tailored AI instructions
    • Enable/disable categories based on your content strategy needs
    • Default categories include content gaps, authority topics, expertise demonstrations, and trust signals
    • Create custom categories for specialized content strategies
  • Priority-based recommendations (high/medium/low)
  • Specific content ideas for each recommendation
  • Dynamic idea generation - "Generate More Ideas" feature for each recommendation
  • Inline editing with autosave for titles, descriptions, and content ideas
  • Delete recommendation cards or individual content ideas
  • Mark content ideas as "Implemented" with optional link to published content
  • CSV export of all content ideas for use in spreadsheets or project management tools
  • Site structure analysis for contextual recommendations
  • Integration with site navigation and sitemap
  • Exportable configuration for multi-site deployments

Post-installation

  1. Configure AI provider settings at /admin/config/ai/providers
  2. Configure permissions at /admin/people/permissions (users need "access ai content strategy" to view reports and "administer ai content strategy" to configure categories and settings)
  3. Configure recommendation categories at /admin/config/ai/content-strategy/categories
  4. Customize global AI settings at /admin/config/ai/content-strategy/settings
  5. Access content strategy recommendations at /admin/reports/ai/content-strategy
  6. Generate initial recommendations using the "Generate Recommendations" button
  7. Use "Generate More Ideas" for additional specific content suggestions
  8. Edit recommendations inline by clicking on any text field (changes autosave)
  9. Mark ideas as "Implemented" and add links to track your content progress
  10. Export your content strategy to CSV for use in other tools

Additional requirements

This module requires:

  • AI module with a configured chat provider

Customization

You can customize the recommendation categories to match your content strategy needs:

  • Add new categories: Create custom recommendation types
  • Modify instructions: Tailor AI analysis per category
  • Enable/disable categories: Control which recommendations appear
  • Reorder categories: Drag and drop to change display order

Categories are configuration entities and can be exported/imported using Drupal's standard configuration management system.

AI Coding Assistant Integration

AI Content Strategy includes a built-in Agent Skills file that teaches AI coding assistants how to generate and manage content recommendations through natural language. Compatible with Claude Code, Codex CLI, Gemini CLI, GitHub Copilot, Cursor, and other tools supporting the standard.

After installing the module, run drush acs:setup-ai to enable AI assistant support. Your AI will then respond to natural language like:

  • "Analyze my site and generate content recommendations"
  • "Show me high-priority content gaps"
  • "Create a new recommendation category for seasonal content"
  • "Mark this idea as implemented and link it to /blog/new-post"
  • "Export all recommendations as JSON"
  • "What content should I create next based on my existing pages?"
